1. КМ трехзвенных МСС

Красная сфера это характерестическая точка, в которую должна 'встать' конечная точка манипулятора при решении ОЗК. 

Подвигай:

- translate x1,
- translate z2,
- rotate x3

Ограничения:

- 20 < translate x1 < 120;
- 20 < translate z2 < 120;
- -90  < rotate x3 < 135;

Отношение:

1:10

- x-> z
- y-> x
- z-> y
